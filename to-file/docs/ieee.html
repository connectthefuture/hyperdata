The Shortest Path to the Future Web

The Web's destination is unclear, but a possible trajectory could be a shift from the Web as a repository for interlinked documents to a more generalized, dynamic system of interlinked data.

  • Published in: IEEE Internet Computing, November 2006

From Here to There

The demands of the Semantic Web vision are generally well-aligned with Web systems' practical requirements. Data-oriented Web systems technologies might help carry us to a Semantic Web sooner rather than later.

  • Published in: IEEE Internet Computing, November 2006

Evolving the Link

The author focuses on the hyperlink, and discusses early criticisms of the Web's most basic - though arguably the most powerful - feature

  • Published in: IEEE Internet Computing, May 2007

A Difficult Abstraction

The developer community is generally overloaded with what they see, leading to simplifications that encourage the development of systems that are inconsistent with underlying specifications.

  • Published in: IEEE Internet Computing, September 2007

Graph Farming

Danny Ayers talks about the Web's evolution, and then offers some suggestions toward another abstraction of the Web that might alleviate many current obstacles to a significantly better Web. Ayers suggests that what we need is a way of thinking about Internet-based systems that supports Tim Berners-Lee's Giant Global Graph abstraction, encompasses processing in this environment yet is conceptually simple enough that real-world developers can use it in their daily work.

  • Published in: IEEE Internet Computing, January 2008

Delivered Deliverables: The State of the Semantic Web, Part 1

Tangible products have emerged from the Semantic Web initiative. The author examines the progress that has been made in regard to specifications, tools, and applications coming that fall under the W3C's initiative.

  • Published in: IEEE Internet Computing, January 2009
